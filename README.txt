Чат-бот "SAPEO" в Telegram 

1. Устанавливаем библеотеку telebot. Для этого в командной строке вводим pip install pyTelegramBotAPI

2. Регистрируем бот через @BotFather в Telegram. Отправляем @BotFather сообщение /newbot. Он запрашивает имя бота. 
Обязательное условие, чтобы имя заканчивалось на bot. Затем @BotFather присылает токен (ключ к боту).

3. Создаём новый проект в PyCharm, в котором создаём 2 файла: 
config.py (здесь создаём одну переменную token, значение которого является токен полученный через @BotFather) 
и bot.py (здесь будет код чат-бота).

4. В bot.py мы импортируем telebot, from telebot import types (чтобы можно было создать кнопки в чат-боте),
from config import token (импортируем ключ к боту, который записали в файле config.py).

5. Создаём bot с использованием импортированного токена.

6. Отдельно создаём названия кнопок (переменная) и её содержание (значение).

7. Для обработки сообщений используем декоратор @bot.message_handler(content_types=['text']). 
Тем самым мы сообщаем, что будем получать текст.

8. Далее создаём метод для обработки сообщений. И с помощью условного оператора задаём, что будет отвечать 
(bot.send_message(message.from_user.id, <текст бота>), если получит сообщение с текстом "Привет", "/help" и 
если сообщение не будет совпадать с данными словами.

9. Создаём кнопки.  keyboard = types.InlineKeyboardMarkup()
Определяем название и значение кнопки (key_rcpeo = types.InlineKeyboardButton(text='Сайт САПЭО', callback_data='rcpeo')).
Добавляем кнопки на экран (keyboard.add(key_rcpeo)).

10. Добавляем декоратор (@bot.callback_query_handler(func=lambda call: True)) и метод, для того, чтобы наши кнопки работали и 
при нажатии определённой кнопки, будет выдаваться ответ, который мы прописывали в значении соответсвующей переменной.

11. Запускаем постоянный чат-бот  (bot.polling(none_stop=True, interval=0)).
